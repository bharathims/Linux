# Tmux

## Ctrl+B cheat sheet

### Session 

* S - list session
* $ - rename current session
* D - detact current session
* ? - hlep

### Window

* C - create new window
* , - rename window
* W - list the windows
* 0 - 9 - switch between windows

### Pane

* % - horizondal split
* " - vertical split
* Q - shown pane numbers
* O - move through pane in order
* X - close pan
* right, left, up, down arros - move through panes
